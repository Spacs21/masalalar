JavaScript Tsikllari

Kirish

Ushbu uy vazifasida JavaScript tilida "for" va "while" tsikllarini o‘rganamiz. Quyida har bir vazifa uchun batafsil tushuntirishlar va misollar keltirilgan.

"For" Tsikl Vazifalari

Vazifa 1

Masala: "for" tsiklidan foydalanib, k qiymatini n marta chop etish.

Yechim: Kodda k o‘zgaruvchisi 5 ga teng va n o‘zgaruvchisi 3 ga teng. Tsikl k ni n marta chop etadi.

let k = 5;
let n = 3;

for (let i = 0; i < n; i++) {
  console.log(k);
}

Vazifa 2

Masala: "for" tsiklidan foydalanib, a dan b gacha bo‘lgan butun sonlarni chop etish.

Yechim: Kodda a o‘zgaruvchisi 2 ga teng va b o‘zgaruvchisi 6 ga teng. Tsikl a dan b gacha bo‘lgan sonlarni chop etadi va necha son borligini hisoblaydi.

let a = 2;
let b = 6;
let count = 0;

console.log("Butun sonlar:");

for (let i = a; i <= b; i++) {
  console.log(i);
  count++;
}

console.log("Chiqarilgan sonlar soni:", count);

Vazifa 3

Masala: "for" tsiklidan foydalanib, b dan a gacha bo‘lgan sonlarni chop etish.

Yechim: Kodda a o‘zgaruvchisi 2 ga teng va b o‘zgaruvchisi 6 ga teng. Tsikl b dan a gacha bo‘lgan sonlarni chop etadi.

let a = 2;
let b = 6;

for (let i = b; i >= a; i--) {
  console.log(i);
}

Vazifa 4

Masala: "for" tsiklidan foydalanib, 1 kg dan 10 kg gacha bo‘lgan har bir kilogramm uchun narxni chop etish.

Yechim: Kodda price o‘zgaruvchisi 2 ga teng. Tsikl 1 kg dan 10 kg gacha bo‘lgan har bir kilogramm uchun narxni chop etadi.

let price = 2;

for (let i = 1; i <= 10; i++) {
  console.log(i + " kg = " + i * price);

Vazifa 5

Masala: "for" tsiklidan foydalanib, 0.1 kg dan 1 kg gacha bo‘lgan har bir kilogramm uchun narxni chop etish.

Yechim: Kodda price o‘zgaruvchisi 2 ga teng. Tsikl 0.1 kg dan 1 kg gacha bo‘lgan har bir kilogramm uchun narxni chop etadi.

let price = 2;

for (let i = 0.1; i <= 1; i += 0.1) {
  console.log(i.toFixed(1) + " kg = " + i * price);

Vazifa 7

Masala: "for" tsiklidan foydalanib, a dan b gacha bo‘lgan sonlarning yig‘indisini hisoblash.

Yechim: Kodda a o‘zgaruvchisi 2 ga teng va b o‘zgaruvchisi 6 ga teng. Tsikl a dan b gacha bo‘lgan sonlarning yig‘indisini hisoblaydi va natijani sum o‘zgaruvchida saqlaydi.

Vazifa 10

Masala: "for" tsiklidan foydalanib, n qiymatining harmonik yig‘indisini hisoblash.

Yechim: Kodda n o‘zgaruvchisi 5 ga teng. Tsikl 1 dan n gacha bo‘lgan harmonik sonlarni yig‘adi va natijani sum o‘zgaruvchida saqlaydi.

"While" Tsikl Vazifalari

Vazifa 1

Masala: "while" tsiklidan foydalanib, A qiymatidan B qiymatini kamaytirish.

Yechim: Kodda A o‘zgaruvchisi 20 ga teng va B o‘zgaruvchisi 6 ga teng. Tsikl A qiymatini B qiymatini kamaytiradi va qolgan qiymatni chop etadi.

let A = 20;
let B = 6;

while (A >= B) {
  A -= B;
}

console.log("Remaining:", A);

Vazifa 2

Masala: "while" tsiklidan foydalanib, A qiymatini B qiymatini kamaytirib, necha marta kamaytirganini hisoblash.

Yechim: Kodda A o‘zgaruvchisi 20 ga teng va B o‘zgaruvchisi 6 ga teng. Tsikl A qiymatini B qiymatini kamaytirib, qancha marta kamaytirganini hisoblaydi.

let A = 20;
let B = 6;
let count = 0;

while (A >= B) {
  A -= B;
  count++;
}

console.log("Segments:", count);

Vazifa 3

Masala: "while" tsiklidan foydalanib, N qiymatidan K ga bo‘lingan qoldiqni hisoblash.

Yechim: Kodda N o‘zgaruvchisi 17 ga teng va K o‘zgaruvchisi 5 ga teng. Tsikl N qiymatini K ga bo‘lib, natijani quotient va qoldiqni chop etadi.

let N = 17;
let K = 5;
let quotient = 0;

while (N >= K) {
  N -= K;
  quotient++;
}

console.log("Quotient:", quotient);
console.log("Remainder:", N);

Vazifa 4

Masala: "while" tsiklidan foydalanib, n qiymatini 3 ning darajasiga teng bo‘lishi uchun tekshirish.

Yechim: Kodda n o‘zgaruvchisi 27 ga teng. Tsikl n qiymatini 3 ning darajasiga tengligini tekshiradi va natijani chop etadi.

let n = 27;
let result = "3 - ning darajasi emas";

while (n > 1) {
  if (n % 3 !== 0) break;
  n /= 3;
  if (n == 1) result = "3 - ning darajasi";
}

console.log(result);

Vazifa 5

Masala: "while" tsiklidan foydalanib, n qiymatining 2 ning darajasiga teng bo‘lishi uchun eng kichik k ni hisoblash.

let n = 16;

  power *= 2;
  k++;

console.log("k =", k);
